# 博客系统 
### 1. 开发环境：Idea+Tomcat8.5+Mysql5
### 2. 项目架构：Spring+SpringMVC+Mybatis+Maven+EasyUI
### 3. 项目描述：给用户分配权限，用户登录后可看到自已已有的权限并更改信息。
